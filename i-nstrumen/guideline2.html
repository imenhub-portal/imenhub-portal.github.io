Walk-in: Start a Session

Click Start on an Active equipment card. The session form opens. Fill in your details and submit.

• Name + Phone *required
• Student/Staff ID — optional
• Affiliation *required
• Sample qty + Material + Duration

5

Booking in Advance

Click Book to open the booking form. Fill in date, slot, personal details, samples, and payment method.

⏰ Date must be ≥ 2 days ahead. If you're a registered PIC, entering your email lifts this limit automatically.
Time slots: Morning (8am–1pm) · Afternoon (2pm–5pm) · Full Day · 2 Days

Check-in on Booking Day

If you have an approved booking for today, clicking Start / Check-in shows an identity verification screen first.

Enter the same phone number you used when booking. Once verified, the session form opens with all fields pre-filled.

Reporting an Issue

Click Report Issue on any equipment card. Two issue types:

Equipment Breakdown → status set to Maintenance
Misconduct / Cleanliness → status set to Attention

Admin Email Gate

Click Control on the home dashboard. You'll be asked for your PIC email.

Only emails registered as a coordinator (in the Config sheet) are accepted. The system unlocks the panel pre-filtered to your assigned lab.

Add or Edit Equipment

Click the edit icon (or "Add Equipment") to open the form. Configure all properties — basic info, status, access mode, calibration, and material/process options.

Tracking unit determines what the user sees as the sample-quantity label (pcs, mg, g, hours).

Process Capability Builder

└─ Variant method (PECVD)
└─ Specifics recipe (SiO₂)
