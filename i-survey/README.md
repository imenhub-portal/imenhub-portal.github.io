# Sistem Kajian Kebolehpasaran Program IMEN UKM

[cite_start]Sistem kaji selidik interaktif ini dibangunkan menggunakan **Google Apps Script**, **Tailwind CSS**, dan **Alpine.js** untuk menilai kebolehpasaran program di IMEN[cite: 1].

## Ciri-Ciri Utama
- [cite_start]**UI/UX Premium**: Menggunakan reka bentuk moden dengan identiti warna UKM[cite: 7].
- [cite_start]**Multi-Step Form**: Memudahkan responden menjawab soalan yang banyak tanpa rasa terbeban[cite: 4].
- **Automasi Data**: Respon disimpan secara automatik ke dalam Google Sheets dan dianalisis secara real-time.
